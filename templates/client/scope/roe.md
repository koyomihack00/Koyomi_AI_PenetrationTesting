# Rules of Engagement

Allowed:
- Authorized testing only
- Passive recon
- Safe active recon
- Service fingerprinting
- TLS/header/cookie review
- Authenticated testing with approved accounts only
- Evidence collection
- Reporting

Ask before:
- Exploitation
- Credential testing
- Brute force
- Password spraying
- Auth bypass testing
- State-changing actions
- Availability-impacting actions

Forbidden:
- Testing outside scope
- DoS / stress testing
- Malware
- Persistence
- Unauthorized credential access
